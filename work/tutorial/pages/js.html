JavaScript

JavaScript zkráceně JS je scriptovací jazyk určen pro tvorbu webů.

Používá se zejména k určení vlastností jednotlivých elementu, v tom smyslu, že například tento kód vytvoří tlačítko, které zobrazí přesný čas.

<!DOCTYPE html>
 <html>
 <head>
 <script>
 function displayDateTime() {
 var currentDateTime = new Date();
 document.getElementById("display").innerHTML = currentDateTime;
 }
 </script>
 </head>
 <body>
 <button onclick="displayDateTime()">Display date and time</button>
 <div id="display"></div>
 </body>
 </html>
