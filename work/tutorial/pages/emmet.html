Emmet

Emmet je sada pluginů, které umožňují rychlejší psaní html kódu. Byl vytvořen v roce 2008 a je stále rozvíjen. Jako začínající web developery vás budou učit klasické psaní html kódu, ale já vám zde ukážu, jak vám může emmet ušetřit spoustu času.

Oba gify znázorňují pravou podstatu pluginu emmet, který šetří spoustu času jak v HTML tak i v CSS.
