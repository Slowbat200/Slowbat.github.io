Cascading Style Sheets

neboli css se ve web developmentu používá na stylizaci HTML kontentu.

Jakákoliv stylizace HTML se zapisuje do css, buď přímo pomoci "tagů", "class" či "id".

HTML se dá stylovat dvěma způsoby.

Buď přímo do HTML

<!DOCTYPE html>
 <html>
 <head>
 <style>
 body{
 background:black;
 }
 </style>
 </head>
 </html>
Vyzkoušej

Nebo do CSS souboru.

*{
 margin:0;
 padding:0;
 box-sizing:border-box;
 }
Vyzkoušej

Aby CSS soubor správně fungoval, musí se v HTML hlavičce uvést externí odkaz na CSS soubor.

<!DOCTYPE html>
 <html>
 <head>
 <link rel="stylesheet" href="style.css" />
 </head>
 </html>
