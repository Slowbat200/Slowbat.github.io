Úvod do HTML

HTML neboli Hypertext Markup Language je značkovací jazyk vytvořený v roce 1990 na tvorbu webových stránek. HTML se dá považovat za kostru celé webové stránky. Jakýkoliv text nebo obrázky, které na webu vydíte jsou napsány pomocí HTML značek.

Úvod do CSS

CSS neboli Cascading Style Sheet je jazyk na stylizaci webových stránek. První verze CSS byla vydána v roce 1996 W3C (World Wide Web Consortium). Slouží k oddělení stylizace od struktury webu.

JS je jediný scriptovací jazyk, který se řadí do front-end rodiny. JS se stará o pohyb stránky, např. animace.
